GPA Scaling

Understand GPA scales by university.

Each university can use a different scale. This page explains how grading scales work and provides a clean reference you can use while setting up your courses.

Add your university How it works

Scale types

Some schools use 4.0, others use 4.3, 4.33, 9.0, or percentage-to-letter conversions. MySemester supports multiple formats.

Letter mappings

Letter grades often map to GPA ranges. These mappings vary by faculty and institution, so always confirm the official chart.

Official sources

Use your university’s official policy page for the most accurate and current GPA conversion rules.
